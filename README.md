# Base Onchain Identity

مشروع تجريبي لبناء **هوية رقمية قائمة على البلوكشين (On-chain Identity)**  
باستخدام React + TypeScript + Vite.

## الفكرة
إنشاء واجهة ويب بسيطة تمثل هوية رقمية يمكن ربطها لاحقًا بـ:
- DID (Decentralized Identifier)
- محافظ Web3
- ملفات `.well-known/did.json`

## التقنيات المستخدمة
- React
- TypeScript
- Vite

## التشغيل محليًا
```bash
npm install
npm run dev
